# 아이디코인
아이디어와 개발자를 이어주는 토큰 이코노미

## 문제점
- 삶에서 필요한 것들에 대해 아이디어는 있지만 직접 그것을 만들 수 없는 사람들이 대부분이다.

## 목차
- 아이디어를 두고 사람들이 배팅을 한다. 
- 배팅된 아이디어중 가장 인기가 많은 아이디어가 채택되면 전문가들이 제작을 한다. 
- 수익에 대해서 각자 나누어 가진다.
